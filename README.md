# Java Swing으로 만든 야구 예매프로그램
- 초기 데이터베이스 및 자바 공부를 위한 프로젝트
- 인터파크의 야구 예매를 참고하여 프로그램 제작
- 로그인, 회원가입, 예매, 예매확인, 예매취소가 가능한 프로그램
- 관리자 모드로 경기 추가 및 제거 가능

# 데이터베이스
- 무료 오픈 소스 관계형 데이터베이스인 postgreSQL을 사용
